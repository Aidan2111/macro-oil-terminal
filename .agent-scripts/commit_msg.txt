feat(quant): top-5 desk review items — cointegration, GARCH, Cushing, crack, extended risk

New modules:
* cointegration.py — Engle-Granger test + ADF on residual + AR(1)-derived
  half-life. Returns "cointegrated" / "weak" / "not_cointegrated" /
  "inconclusive" + hedge ratio (β) + alpha. rolling_engle_granger()
  slides the test for structural-break spotting.
* vol_models.py — GARCH(1,1) wrapper via `arch`. Defensive: returns
  ok=False on fit failure or short series, never raises. Exposes
  conditional σ, Z, persistence (α+β).
* crack_spread.py — pulls RB=F + HO=F + CL=F via yfinance, computes
  3-2-1 crack in USD/bbl (2·RBOB + HO)/3 × 42 − WTI, plus 30d rolling
  correlation vs Brent-WTI. Graceful ok=False on offline.

quantitative_models.py:
* compute_spread_zscore now emits Spread_EwmaStd + Z_Vol alongside the
  rolling-window Z_Score. λ=0.94 RiskMetrics EWMA — reacts to regime
  change faster than the rolling std.
* backtest_zscore_meanreversion now returns sortino, calmar, var_95,
  es_95, rolling_12m_sharpe. Empty-frame branch seeds all keys to 0.

providers/_eia.py:
